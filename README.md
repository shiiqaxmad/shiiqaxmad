## ⚙️ HOW TO USE

1. Click **GET PAIR CODE**
2. Enter your number (Example: 25261xxxxxxx)
3. Copy the code
4. Open WhatsApp
5. Go to **Linked Devices**
6. Click **Link with phone number**
7. Paste code ✅

---

## ⚠️ IMPORTANT NOTES

- ❗ Code works only for **30 seconds**
- ❗ Use correct format: `252xxxxxxxxx`
- ❗ If failed, generate new code
- ❗ If Render sleep → use QR
